# StreamingAssets — Terrain Data

This folder holds the real DEM terrain data generated by the Python pipeline.

## How to Generate

```bash
cd mvp
uv run mvp --sample --skip-render   # synthetic terrain (fast, no download)
uv run mvp --skip-render             # real Copernicus DEM (requires internet)
```

Then copy the output files here:

```bash
cp mvp/output/terrain.raw Assets/StreamingAssets/
cp mvp/output/terrain_metadata.json Assets/StreamingAssets/
```

## Files

| File | Purpose |
|------|---------|
| `terrain.raw` | 16-bit RAW heightmap (513x513, little-endian) |
| `terrain_metadata.json` | Terrain dimensions, elevation range, BBOX |

## How It Works

`TerrainGenerator.cs` automatically checks `StreamingAssets/terrain.raw` on startup.
If found, it loads the real DEM data and applies the metadata for correct world sizing.
If not found, it falls back to synthetic terrain generation.
